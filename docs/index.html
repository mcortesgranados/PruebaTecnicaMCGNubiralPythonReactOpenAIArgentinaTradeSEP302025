🌎 Argentina Trade Insights: Explorador Inteligente de Importaciones y Exportaciones 🤖

Autora : Manuela Cortes Granados

Repo GitHub : https://github.com/mcortesgranados/PruebaTecnicaMCGNubiralPythonReactOpenAIArgentinaTradeSEP302025/commits?author=mcortesgranados

Objetivo: Construir una aplicación web fullstack moderna que permita explorar, consultar y analizar datos abiertos de importación/exportación de Argentina, potenciando insights mediante IA generativa (ChatGPT, Gemini, Amazon Q, etc.) para responder preguntas complejas, generar reportes y visualizaciones interactivas.
Enfocado en mostrar skills de arquitectura limpia, frontend moderno, backend escalable, cloud e integración IA.

🧩 Arquitectura & Stack

📁 Frontend (React + TypeScript)
UI moderna, responsiva, con filtros y visualizaciones (charts, dashboards)
Componente de chat para preguntas en lenguaje natural (integrado con backend e IA). Generación automática de tooltips/descripciones vía IA.
📁 Backend (Python/FastAPI)
APIs RESTful, microservicios de ingestión/procesamiento de datos y consultas IA.
Servicio de integración con APIs de IA generativa (OpenAI, Gemini, Amazon Q, etc.)
📁 Infraestructura Cloud
Docker, scripts de despliegue, integración continua (GitHub Actions), docs para AWS/Azure.
📁 Observabilidad
Logs estructurados, métricas básicas, monitoreo (simulado/local o real en cloud).
📄 Bases de Datos
PostgreSQL (principal), Redis (caching).
📄 Mensajería
Ejemplo de integración con RabbitMQ para procesamiento batch o notificaciones.
📄 Auth
JWT simple, autenticación simulada para demo.
📄 Documentación
Docs técnicas en Notion/Confluence o Markdown, decisiones de arquitectura y comparativas IA/cloud.

🚀 Funcionalidades Clave

  • Visualización interactiva 📊: Exploración de importaciones/exportaciones por sector, año, provincia, etc. con filtros avanzados y gráficos.
  • Chat de IA generativa 💬🤖: Permite preguntar en lenguaje natural (ej: "¿Qué sectores lideraron las exportaciones en 2023?"). Responde usando análisis sobre los datasets. Prompt engineering en backend.
  • Generador de reportes automáticos 📄✨: Descarga de informes PDF/Excel con gráficos y explicaciones generadas por IA.
  • Documentación técnica 📝: Decisiones de arquitectura, comparativas entre plataformas IA/cloud, guía de evolución/modularidad.
  • Uso de IA generativa como copiloto 👩‍💻🤝🤖: Documentar cómo se usó Copilot, ChatGPT, Gemini, etc. en el desarrollo, documentación y testing.
